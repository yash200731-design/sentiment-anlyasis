# 🧠 AI Sentiment Analysis

An AI-powered web application that analyzes user reviews and predicts whether the sentiment is **Positive 😊**, **Negative 😞**, or **Neutral 😐** using Machine Learning and Natural Language Processing (NLP).

## 🚀 Live Demo

🌐 Live Website: https://sentiment-anlyasis-jy7i4d4ih-yash-s2007-projects.vercel.app/

## 📌 Features

- 🔍 Analyze text sentiment in real time
- 😊 Predicts Positive, Negative, and Neutral sentiments
- 🤖 Machine Learning-based prediction
- 📊 Natural Language Processing (NLP)
- 📱 Responsive and user-friendly interface
- ⚡ Fast prediction with a trained ML model

## ⚙️ How It Works

1. User enters a review.
2. The text is cleaned and preprocessed.
3. TF-IDF converts the text into numerical features.
4. The trained Logistic Regression model predicts the sentiment.
5. The result is displayed instantly.

## 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/yash200731-design/sentiment-anlyasis.git
```

### Navigate to the project

```bash
cd sentiment-anlyasis
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the backend

```bash
python app.py
```

### Open the frontend

Open `index.html` in your browser.
